# 卡咔桌面操作功能 - 快速上手指南

## 什么是桌面操作功能？

卡咔现在可以在你的电脑上帮你做一些事情啦！

**目前支持**：
- 📝 在桌面创建文件和小纸条
- 🔜 截图分享（开发中）
- 🔜 播放提醒音效（开发中）

---

## 快速开始

### 步骤 1：启动卡咔核心服务

打开命令行，进入卡咔项目目录：

```bash
cd D:\Python\AgentRobot\kaka-v2
```

激活虚拟环境并启动 kaka-core：

```bash
.venv\Scripts\activate
cd services\kaka-core
uvicorn kaka_core.api.app:app --host 0.0.0.0 --port 8001
```

看到 `Application startup complete` 就成功了！

### 步骤 2：启动本地组件

**新开一个命令行窗口**，同样进入项目目录：

```bash
cd D:\Python\AgentRobot\kaka-v2
.venv\Scripts\activate
cd apps\kaka-local
python src\main.py
```

看到这样的输出就成功了：
```
==================================================
  卡咔本地组件
==================================================
卡咔本地组件已启动，连接到 http://127.0.0.1:8001
轮询间隔: 3 秒
```

### 步骤 3：在 QQ 群试试

向卡咔发消息：

```
@卡咔 在桌面写个文件，内容是今天要早睡
```

卡咔会回复：
```
好的，我来写~
```

几秒后：
```
写好了~ 已创建 小纸条.txt
```

**检查桌面**：
你会在桌面看到一个新文件夹 `卡咔的小角落/`，里面有 `小纸条.txt`，内容是"今天要早睡"！

## 安全说明

**放心使用，卡咔很安全**：

✅ **只能在指定文件夹操作**
- 只能在 `桌面/卡咔的小角落/` 和 `文档/卡咔/` 创建文件
- 不会动你的其他文件

✅ **文件类型受限**
- 只能创建 `.txt`, `.md`, `.json`, `.log` 文件
- 不能运行程序、删除文件

✅ **可以随时删除**
- 不喜欢的文件直接删除就好
- 不想要功能了，关闭 kaka-local 即可

## 常见问题

### Q: 为什么卡咔说"写好了"但桌面没有文件？

**A**: 检查本地组件（kaka-local）是否在运行。如果没运行，任务会堆积在数据库，等本地组件启动后才会执行。

### Q: 可以让卡咔删除文件吗？

**A**: 不行，为了安全，卡咔只能创建文件，不能删除或修改现有文件。

### Q: 群友都能让卡咔在我电脑上操作吗？

**A**: 目前只要是合理请求都会执行。后续会加入 LLM 决策，让卡咔判断要不要执行。

### Q: 本地组件必须一直开着吗？

**A**: 是的。关闭后任务会堆积，重新启动时会执行所有待办任务。

### Q: 可以开机自启吗？

**A**: 目前需要手动启动。后续版本会支持打包成 .exe 和开机自启。

## 关闭功能

如果不想用了：

1. **关闭本地组件**：在运行 kaka-local 的命令行按 `Ctrl+C`
2. **保持 kaka-core 运行**：其他功能（聊天、记忆）不受影响
